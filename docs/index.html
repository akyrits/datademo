PawSense AI: Real-Time Dog Emotion Detection

Detecting happy vs. stressed dogs from images using AlexNet and transfer learning.

Problem

Dogs often show subtle signs of stress or anxiety—pacing, lip licking, pinned ears, or avoiding eye contact—long before humans notice a “behavior problem.” By the time we see barking, destruction, or withdrawal, the dog may have been uncomfortable for a long time.

At the same time, pet cameras and smart collars are becoming common, but they mostly track where the dog is, not how the dog feels. There is an opportunity to use computer vision to support better, earlier detection of stress.

Motivation

  • Animal welfare: Help owners catch stress early and respond with training or environment changes.
  • Practical deployment: A lightweight CNN could run on existing pet cameras or apps.
  • Learning goal: Apply transfer learning with AlexNet to a real-world dataset.

This project explores whether a fine-tuned AlexNet can reliably distinguish happy vs. stressed/anxious dogs from still images.

Project Scope

  • Task: binary image classification – happy vs. stressed dogs.
  • Input: single RGB dog image resized to 224×224.
  • Output: predicted class + confidence score.
