PawSense AI: Detecting Stress & Emotional States in Dogs

A CNN-based approach using transfer learning with AlexNet.

Problem

Dogs frequently display subtle signs of stress—pacing, lip licking, pinned ears, whale eye, a stiff posture—long before humans perceive a “behavior problem.” Many dog bite and shelter intake cases are preceded by unrecognized stress signals.

Pet tech (cameras, collars, trackers) is increasing in popularity, but these systems measure location and activity, not emotional state. A vision-based stress detection system could help owners respond earlier and promote stronger human–animal relationships.

Motivation

  • Improve early detection of stress, preventing escalation.
  • Support dog welfare and behavioral health.
  • Enable real-time monitoring through pet cameras and apps.
  • Apply deep learning skills to a meaningful real-world problem.

Project Scope

  • Binary classification: Happy vs Stressed/Anxious.
  • Input: single image resized to 224×224.
  • Output: predicted class + confidence score.
  • Model: transfer learning with pretrained AlexNet.

The technical details of the AlexNet architecture and training pipeline are documented on the AlexNet Approach page.
